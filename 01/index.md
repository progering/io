## Kilpkonn
```python
from turtle import *

# liigu koos joonistamisega
forward(100) # liigu 100 sammu
left(90)     # pööra 90 kraadi
forward(100)
left(90)
forward(100)
left(90)
forward(100)
left(90)

# liigu kõrvale ilma joonistamata
```
